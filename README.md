# Sliding-Puzzle-Solver-GUI
Implementation of A* Search algorithm to solve 8 tile sliding puzzle.

## Required Package: Tkinter
This code is only interactable through the UI. To test it, ensure you have the tkinter python package on your
computer.
<br>Use the following command depending on your system:
|Windows / MacOS (Intel)| Linux MacOS |(Apple Silicon)|
|-----|-----|-----|
pip install tk|sudo apt-get install python-tk|install homebrew (https://brew.sh/) and use brew install python-tk|
<br>
Then you can execute the code using "python 8Puzzle.py"
A window should pop up with the GUI
